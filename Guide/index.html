I. Tâches Annuelles

1. Mise à jour des tarifs

  • Fichier : index.html
  • Étapes :
    1. Ouvrir index.html
    2. Rechercher tarifs-container
    3. Modifier les prix dans <div class="tarif-price">
    4. Mettre à jour les descriptions dans <ul class="tarif-features">

2. Changement des horaires

  • Fichier : index.html
  • Étapes :
    1. Localiser la section <section id="horaires">
    2. Modifier les créneaux dans les balises <p> sous chaque <h3>

3. Mise à jour des statistiques

  • Fichier : script.js
  • Étapes : 
    const CONFIG = {
  stats: {
    licencies: 125,
    jeunes: 86,
    animateurs: 6
  }
};

II. Tâches Hebdomadaires

1. Ajouter une actualité

{
  "date": "AAAA-MM-JJ",
  "titre": "Titre de l'actualité",
  "description": "Description complète...",
  "image": "chemin/vers/image.jpg",
  "lien": "https://lien-externe.com"
}

Tips :

  • Placer les nouvelles actualités en haut du fichier
  • Les images doivent être ajoutées dans images/Actu/

2. Ajouter un événement à venir

{
  "date": "AAAA-MM-JJ",
  "titre": "Nom de l'événement",
  "description": "Détails...",
  "image": "chemin/vers/image.jpg",
  "lien": "https://lien-inscription.com"
}

3. Ajouter des archives photos

{
  "year": "2025",
  "events": [
    {
      "title": "Nom de l'événement",
      "date": "JJ Mois AAAA",
      "description": "Description...",
      "images": ["chemin/photo1.jpg", "chemin/photo2.jpg"]
    }
  ]
}
  • Important :
  • Créer un nouveau dossier dans images/Tournoi_AAAA/ pour les photos
  • Utiliser le même nom de dossier dans le JSON

III. Ajouter une Nouvelle Année

Pour les archives photos :

  1. Ouvrir archives-photos.json
  2. Ajouter un nouveau bloc en haut : 
    {
  "year": "2026",
  "events": []
}

Pour le filtre d’année :

  • Fichiers : archives-actualites.html et archives-photos.html
  • Étapes : 
    <button class="archive-year-btn" data-year="2026">2026</button>

IV. Bonnes Pratiques

  • Validation des liens :
    • Tester tous les liens après modification
    • Utiliser des URL complètes (https://...)
  • Gestion des images :
    • Taille max : 1MB par image
    • Formats acceptés : JPG, PNG
    • Nommer les fichiers sans espaces (ex: tournoi_noel_2025.jpg)
  • Sauvegardes :
    • Toujours faire une copie des fichiers avant modification
    • Utiliser des commentaires dans le JSON pour se repérer : 
      // 2025 - Tournoi d'été
{ ... }

V. Dépannage Rapide
